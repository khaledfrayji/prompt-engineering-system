# 🏗️ Build a Skill Workflow

## 🎯 Purpose

This workflow shows how to turn an idea, use case, or weak prompt into a production-ready AI skill.

It connects the core modules of the Prompt Engineering System into one practical process.

---

## 🧠 When to Use

Use this workflow when:
- building a new AI skill from scratch  
- turning a vague prompt into a structured system  
- improving a prompt before production  
- creating reusable AI workflows  

---

## ⚙️ Workflow Overview

```text
Define Goal
   ↓
Build Skill
   ↓
Lock Output Format
   ↓
Stress Test
   ↓
Fix Failures
   ↓
Strengthen Constraints
   ↓
Evaluate Reliability
   ↓
Optimize Language
   ↓
Add Feedback Loop (Optional)
   ↓
Production-Ready Skill
