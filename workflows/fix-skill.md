# 🛠️ Debug a Skill Workflow

## 🎯 Purpose

This workflow helps you diagnose, analyze, and fix failing or inconsistent AI skills.

It focuses on identifying root causes and applying targeted fixes — not patching symptoms.

---

## 🧠 When to Use

Use this workflow when:
- a skill produces inconsistent outputs  
- results vary across inputs  
- edge cases break the system  
- structure or formatting is unstable  
- quality degrades in real-world usage  

---

## ⚙️ Workflow Overview

```text
Identify Failure
   ↓
Stress Test
   ↓
Log Failures
   ↓
Rank Severity
   ↓
Fix Root Causes
   ↓
Strengthen Constraints
   ↓
Lock Output Format
   ↓
Re-Test
   ↓
Evaluate Reliability
   ↓
Production-Ready Fix
